# Layout –∫–ª–∞—Å—Å—ã

–ö–∞–∂–¥—ã–π –∫–ª–∞—Å—Å layout –Ω–∞—á–∏–Ω–∞–µ—Ç—Å—è —Å `f` –∞ –¥–∞–ª—å—à–µ —Å–ª–µ–¥—É—é—â–∏–µ —á–∞—Å—Ç–∏ –ø–æ–¥—Å—Ç–∞–≤–ª—è—é—Ç—Å—è –∫–∞–∫:

### `f{flexDirection}{justifyContent}{alignItems}`

### 1. flex-direction

* r ‚Üí row
* c ‚Üí column

### 2. justify-content

* c ‚Üí center
* s ‚Üí start
* e ‚Üí end
* b ‚Üí between
* a ‚Üí around
* ev ‚Üí evenly
* st ‚Üí stretch
* '' ‚Üí –ü—É—Å—Ç–æ (–Ω–µ —É–∫–∞–∑—ã–≤–∞–µ—Ç—Å—è)

### 3. align-items

* c ‚Üí center
* s ‚Üí start
* e ‚Üí end
* b ‚Üí baseline
* st ‚Üí stretch
* '' ‚Üí –ü—É—Å—Ç–æ (–Ω–µ —É–∫–∞–∑—ã–≤–∞–µ—Ç—Å—è)

–¢–∞–∫ `st` –∏ `ev` –∑–Ω–∞—á–µ–Ω–∏—è –º–æ–≥—É—Ç —Å–±–∏–≤–∞—Ç—å —Å —Ç–æ–ª–∫—É –º—ã –∏—Ö –≤—ã–¥–µ–ª—è–µ–º –≤ –∫–ª–∞—Å—Å–µ –∑–Ω–∞–∫–∞–º–∏ `-`, –ø–æ –∏—Ç–æ–≥—É —É –Ω–∞—Å –ø–æ–ª—É—á–∞—é—Ç—Å—è –≤–æ—Ç —Ç–∞–∫–∏–µ
–∫–ª–∞—Å—Å—ã:
`fr` `fc` `fcsc` `frsc` `fce` `fc-st` `fr-ev-c` –∏ —Ç.–¥.

–¢–∞–∫ –º–æ–∂–Ω–æ —Å–≥–µ–Ω–µ—Ä–∏—Ä–æ–≤–∞—Ç—å –≤—Å–µ –≤–æ–∑–º–æ–∂–Ω—ã–µ –∫–æ–º–±–∏–Ω–∞—Ü–∏–∏:

```python
done_classes = set()
for i in ('r', 'c'):
    for j in ('c', 's', 'e', 'b', 'a', 'ev', 'st', ''):
        for k in ('c', 's', 'e', 'b', 'st', ''):
            cls = f'f{i}'
            if not j:
                done_classes.add(cls)
                continue
            if j in ('ev', 'st'):
                cls += f'-{j}-'
            else:
                cls += j
            if not k:
                done_classes.add(cls.replace('--', '-').removesuffix('-'))
                continue
            if k == 'st':
                cls += f'-{k}-'
            else:
                cls += k
            done_classes.add(cls.replace('--', '-').removesuffix('-'))

print(f'–ö–æ–ª–∏—á–µ—Å—Ç–≤–æ –∫–ª–∞—Å—Å–æ–≤: {len(done_classes)}')
done_classes = sorted(done_classes, key=lambda x: (len(x), x))
print('–ü–æ–ª—É—á–µ–Ω–Ω—ã–µ –∫–ª–∞—Å—Å—ã:')
print(*done_classes, sep='\n')
```
